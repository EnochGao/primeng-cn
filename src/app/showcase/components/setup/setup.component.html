开始

PrimeNG是一组丰富的开源原生Angular UI组件库。

视频教程

观看视频教程，了解本指南中所记录的步骤。

下载

PrimeNG可用于npm中，如果你有一个现有的应用程序，运行以下命令将其下载到你的项目中。

npm install primeng --save npm install primeicons --save
导入

UI组件被配置为模块，一旦PrimeNG被下载和配置，模块和api就可以从你的应用程序代码中以primeng/{module}简写形式导入。每个组件的文档都写明了导入路径。

import {AccordionModule} from 'primeng/accordion'; //手风琴和手风琴tab import {MenuItem} from 'primeng/api'; //api
样式表

css依赖如下，Prime的图标，你选择的主题和组件的结构css。

在angular.json文件中找到 styles 数组并且添加如下内容:

... "styles": [ "node_modules/primeicons/primeicons.css", "node_modules/primeng/resources/themes/lara-light-blue/theme.css", "node_modules/primeng/resources/primeng.min.css", ... ]

涟漪

涟漪是受支持组件(如按钮)的可选动画。它在默认情况下是禁用的，需要在你的主组件中全局启用，例如 app.component.ts中通过注入 PrimeNGConfig使用

import { PrimeNGConfig } from 'primeng/api'; @Component({ selector: 'app-root', templateUrl: './app.component.html' }) export class AppComponent implements OnInit { constructor(private primengConfig: PrimeNGConfig) {} ngOnInit() { this.primengConfig.ripple = true; } }
动画

各种组件利用angular动画来改善用户体验，从angular 4开始，动画有自己的模块，所以你需要将BrowserAnimationsModule导入到你的应用中。如果你喜欢全局禁用动画，导入NoopAnimationsModule代替它。

npm install @angular/animations --save import {BrowserModule} from '@angular/platform-browser'; import {BrowserAnimationsModule} from '@angular/platform-browser/animations'; @NgModule({ imports: [ BrowserModule, BrowserAnimationsModule, //... ], //... }) export class YourAppModule { }
Angular CLI 集成

Angular CLI是Angular的官方CLI工具。我们强烈建议在启动Angular项目时使用Angular CLI。

依赖项

添加primeng和primeicons作为依赖项。

"dependencies": { //... "primeng": "^14.0.0-rc.1", "primeicons": "^5.0.0" },
样式配置

在样式部分配置所需的样式，下面的例子使用Saga Blue主题。

"styles": [ "node_modules/primeng/resources/themes/lara-light-blue/theme.css", "node_modules/primeng/resources/primeng.min.css", "node_modules/primeicons/primeicons.css", //... ],
