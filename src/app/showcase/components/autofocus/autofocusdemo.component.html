AutoFocus

AutoFocus管理可聚焦元素在加载时的焦点.

导入
import {AutoFocusModule} from 'primeng/autofocus';
开始使用

在初始加载时，自动对焦应用于任何可对焦的输入元素。它默认是禁用的，需要手动启用.

<input pAutoFocus [autofocus]="true" > export class AutoFocusDemo { }
属性
名称 类型 默认值 描述
autofocus boolean null 当出现时，它指定组件应该自动获得焦点.
事件

指令没有事件.

样式

指令不会对host应用任何样式.

依赖

无.
