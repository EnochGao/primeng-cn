Toolbar

Toolbar is a grouping component for buttons and other content.
引入
import {ToolbarModule} from 'primeng/toolbar';
准备开始

Toolbar provides start, center and end templates to place content at these sections.

<p-toolbar> <div class="p-toolbar-group-start"> <p-button label="New" icon="pi pi-plus"></p-button> <p-button label="Upload" icon="pi pi-upload" styleClass="p-button-success"></p-button> <i class="pi pi-bars"></i> <p-splitButton label="Save" icon="pi pi-check" [model]="items" styleClass="p-button-warning"></p-splitButton> </div> <div class="p-toolbar-group-end"> <p-button icon="pi pi-search"></p-button> <p-button icon="pi pi-calendar" styleClass="p-button-success"></p-button> <p-button icon="pi pi-times" styleClass="p-button-danger"></p-button> </div> </p-toolbar>
Properties
名称 类型 默认值 描述
style string null Inline style of the component.
styleClass string null Style class of the component.
样式

Following is the list of structural style classes, for theming classes visit theming page.

名称 元素
p-toolbar Main container element.
p-toolbar-group-start Start content container.
p-toolbar-group-center Center content container.
p-toolbar-group-end End content container.
模板
名称 参数
start -
center -
end -
依赖

None.
